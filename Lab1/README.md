# Lab Work №1 / Лабораторна робота №1

This directory contains the implementation of Laboratory Work №1, which focuses on setting up the development environment and creating initial projects for the PSoC 4 platform.

---

## English Version

### 1. About the Lab
The goal of this laboratory work is to get acquainted with the functional capabilities of the PSoC Creator 4.4 IDE and the Saleae Logic analyzer. Key tasks include:
* Installing and configuring the necessary software for PSoC development.
* Learning the workflow of creating, debugging, and executing projects.
* Exploring the CapSense technology using built-in code examples.
* Visualizing digital signals and analyzing basic communication protocols.

### 2. Tools and Technologies
* **IDE:** PSoC Creator 4.4.
* **Hardware:** PSoC 4200 (PSoC 4 Pioneer Kit).
* **Analysis Tool:** Saleae Logic analyzer (for digital signal capture).
* **Key Features:** CapSense CSD (Capacitive Sensing), UART/SPI/I2C protocol decoding.

### 3. Implementation Details
* **Environment Setup:** Installed PSoC Creator and Logic software following the provided technical instructions.
* **Project Creation:** Created a project based on the `CapSense_CSD_P4_Design` code example to study touch sensing capabilities.
* **Workflow:** Practiced the full development cycle: Writing/Configuring code -> Build (Shift+F6) -> Compile (Ctrl+F6) -> Programming the target device (Ctrl+F5).
* **Verification:** Used the Logic analyzer to capture and decode waveforms, ensuring the microcontroller interacts correctly with peripheral components.

## Українська версія

### 1. Про що лабораторна робота
Метою цієї роботи є ознайомлення з функціональними можливостями середовища PSoC Creator 4.4 та логічного аналізатора Saleae Logic. Основні завдання включають:
* Встановлення та налаштування необхідного ПЗ для розробки на базі PSoC.
* Вивчення циклу створення, відлагодження та виконання проєктів.
* Дослідження технології CapSense на основі вбудованих прикладів коду.
* Візуалізація цифрових сигналів та аналіз базових протоколів зв'язку.

### 2. За допомогою чого зроблена
* **Середовище розробки:** PSoC Creator 4.4.
* **Апаратне забезпечення:** Мікроконтролер серії PSoC 4200 (PSoC 4 Pioneer Kit).
* **Інструменти аналізу:** Логічний аналізатор Saleae Logic (для захоплення цифрових сигналів).
* **Ключові технології:** CapSense CSD (ємнісне сенсорне керування), декодування протоколів UART/SPI/I2C.

### 3. Як зроблена (реалізація)
* **Налаштування середовища:** Встановлено PSoC Creator та ПЗ Logic згідно з технічними інструкціями.
* **Створення проєкту:** На базі прикладу `CapSense_CSD_P4_Design` створено проєкт для вивчення роботи сенсорних кнопок та слайдерів.
* **Процес розробки:** Опрацьовано повний цикл роботи з пристроєм: Конфігурація -> Збірка (Shift+F6) -> Компіляція (Ctrl+F6) -> Прошивка мікроконтролера (Ctrl+F5).
* **Верифікація:** Використано логічний аналізатор для зняття часових діаграм та перевірки коректності роботи цифрових шин і сигналів.
